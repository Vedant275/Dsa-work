## Using SSH for GitHub
To push changes securely using SSH:
1. Generate SSH keys and add your public key to GitHub.
2. Set your remote URL to SSH:
   ```sh
   git remote set-url origin git@github.com:Vedant275/Dsa-work.git
   ```
3. Push your changes:
   ```sh
   git push
   ```
